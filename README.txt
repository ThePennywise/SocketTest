Por favor, bote em mente que eu costumo usar Netty, porém, neste projeto estava querendo fazer livre de quaisquer FrameWorks ou
dependências.

Este projeto é sobre um servidor Back-End que aceita conexões TCP/IP (TCP OVER IP) para receber dados.

Estes dados são chamados de Packets por serem muito pequenos...

Neste programa tem registrado apenas PING_RESPONSE e PING_REQUEST, porém é possível adicionar tudo o que quiser, já que
o sistema todo está criado... Vai da imaginação do programador.
